# Rubyistokei-GTK

This is a GTK+ frontend for [Rubyistokei](http://rubyistokei.herokuapp.com/).

## Use

    % git clone https://github.com/kou/rubyistokei-gtk.git
    % cd rubyistokei-gtk
    % bundle install
    % ruby rubyistokei-gtk.rb
